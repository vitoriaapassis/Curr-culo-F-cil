# 🌸 Currículo Fácil 💼

✨ Uma aplicação desenvolvida em Python para auxiliar estudantes e jovens profissionais na criação de currículos de forma simples, rápida e organizada.

---

## 💖 Funcionalidades

✅ Cadastro de informações pessoais

✅ Objetivo profissional

✅ Formação acadêmica

✅ Cursos e certificações

✅ Projetos e experiências

✅ Geração automática de currículo em PDF

---

## 🛠️ Tecnologias Utilizadas

🐍 Python

🎨 Streamlit

📄 ReportLab

🌿 Git

💻 GitHub

---

## 🚀 Como Executar

### 1️⃣ Instale as dependências

```bash
pip install -r requirements.txt
```

### 2️⃣ Execute a aplicação

```bash
python -m streamlit run app.py
```
